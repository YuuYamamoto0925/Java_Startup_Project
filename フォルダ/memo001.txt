【GitHubのご利用】
<初回やること>
・ユーザ名とメールアドレスの登録
  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"

  (設定確認)
  git config --global --list

<プロジェクト登録時にやること>
・gitフォルダの作成
  git init

<毎回コミットとプッシュ時にやること>
・GitHubへのコミットとプッシュ
  git add .
  git commit -m "Sample_COMENT"
  git push -u origin master

<変更の取り消し>
・変更の取り消し(修正したソースをGitHubの内容のものに戻す)
  (例)
  git checkout -- E:\15プログラム学習\VSCode\Java\Java_Startup_Project\src\Sample001
